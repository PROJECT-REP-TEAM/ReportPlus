## 支持图表类型
- 饼图 `pie`
- 圆环图 `ring`
- 线图 `line`（直线、曲线）
- 柱状图 `column`（分组、堆叠、温度计）
- 区域图 `area`（直线、曲线）
- 雷达图 `radar`
- 圆弧进度图 `arcbar`
- 仪表盘 `gauge`
- K线图  `candle`
- 混合图 `mix`（支持 Point、Line 直线曲线、Column、Area 直线曲线）
- 玫瑰图 `rose`（面积模式、半径模式）
- 词云图 `word`（横向、横纵混排）
- 漏斗图 `funnel`
- 地图 `map`
